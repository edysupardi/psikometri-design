Psikometri adalah aplikasi asesmen psikotes/psikologi online.

**User Types:**
- User publik: register mandiri → verifikasi email → login → asesmen → bayar untuk lihat hasil
- User B2B: perusahaan yang assign asesmen ke kandidat (e.g. recruitment)
- Psikolog: review & approval hasil asesmen (terhimpun dalam HIMPSI)

**Business Model:** Freemium — asesmen gratis, bayar untuk lihat hasil

**Why:** HIMPSI (Himpunan Psikologi Indonesia) dengan warna khas #5F197B menjadi anchor brand identity. Aplikasi harus terasa premium dan trustworthy untuk membangun kepercayaan user publik dan B2B.

**How to apply:** Semua design decisions harus mempertimbangkan ketiga tipe user. Landing page fokus conversion user publik. Dashboard harus accommodate multi-role navigation.
